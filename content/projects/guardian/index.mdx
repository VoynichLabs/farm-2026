## What It Does Today

Farm Guardian watches the property through three cameras and streams live video to a local dashboard. It runs on a Mac Mini M4 Pro sitting on the desk next to the chick brooder — no cloud, no subscriptions, no data leaving the network.

**What's working right now:**

- Live MJPEG streaming from all three cameras to a web dashboard
- PTZ control on the Reolink — pan, tilt, preset positions, spotlight, siren
- Camera presets for one-tap positioning (sky-watch, coop door, yard sweep)
- 4K HTTP snapshots on demand
- Discord alerts with snapshot attachments
- Step-and-dwell patrol mode (11 positions, 30-degree intervals)
- Sky-watch mode — fixed overhead position for hawk surveillance
- USB camera on the Mac Mini monitor pointed into the chick brooder
- Full REST API for remote camera control over a Cloudflare tunnel
- eBird integration polling for regional raptor sightings

**What's in progress:**

- YOLOv8 object detection pipeline (running, being tuned)
- GLM vision model for species-level ID (hawk vs. chicken)
- Automated deterrent escalation (spotlight, siren, audio)
- Animal visit tracking and pattern analysis
- Daily intelligence reports

The detection and deterrence pipeline exists in code but hasn't been battle-tested yet. We're building it out as we go. The priority right now is reliable camera coverage and getting the monitoring solid before we trust the AI to make decisions on its own.

## The Story

This started because hawks were taking chickens. In the first week of April 2026, we lost four birds — including Birdgit, a Speckled Sussex who followed you around like a dog. A hawk took her on April 8th.

The response was immediate. Within hours, Claude Opus built sky-watch mode — locking the PTZ camera onto a fixed overhead position for continuous hawk surveillance. Alert images were upgraded from blurry 1080p RTSP frame grabs to sharp 4K HTTP snapshots.

Reinforcements arrived the same week: 22 new chicks in the brooder, including turkey poults. The flock is rebuilding, and Guardian is watching.

## Built by Claude

Every line of Guardian — from RTSP capture to the web dashboard to the PTZ preset system — was written by Claude in conversation with the farmer. No templates, no boilerplate, no copy-paste from Stack Overflow. A farmer described what he needed, and an AI built it.

The entire codebase is at v2.11.0. Fifteen Python modules, a FastAPI dashboard, SQLite storage, and a REST API that lets a remote Claude session control the cameras from anywhere in the world.
