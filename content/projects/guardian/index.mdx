## The Problem

Hawks and ground predators — fox, raccoon, bobcat, coyote — are killing chickens on a 13-acre rural property in eastern Connecticut. The cameras have built-in spotlight and motion detection, but they can't tell a chicken from a hawk. Guardian adds that intelligence and automates the response.

## The Solution

Farm Guardian is a Python service running on a Mac Mini M4 Pro (64 GB) that watches two cameras on the local network: a Reolink E1 Outdoor Pro PTZ camera covering the yard, and a Samsung Galaxy S7 phone mounted inside the nesting box. It detects predator animals using YOLOv8 + a local vision model for species-level identification, automatically fires camera deterrents (spotlight, siren, audio alarm) on the Reolink, tracks animal visits over time, and sends Discord alerts with snapshots.

No cloud services. No subscriptions. No data leaves the local network except Discord notifications.

## Detection Pipeline

1. **RTSP Capture** — continuous 1 fps stream from each camera. Reolink connects via ONVIF discovery (TCP transport); the S7 uses a direct RTSP URL override (UDP transport) since it doesn't support ONVIF. Per-camera transport is configured independently.
2. **YOLOv8 Detection** — object detection on every frame with MPS acceleration on Apple Silicon
3. **GLM Vision Refinement** — ambiguous classes ("bird") refined to species level ("hawk" or "chicken") by the local vision model
4. **False-Positive Suppression** — minimum bounding box size, dwell time thresholds, confidence floors, zone masking
5. **Visit Tracking** — detections grouped into visits, outcomes tracked per species

## Automated Deterrence

Four escalation levels, triggered per-species:

| Level | Action | Example trigger |
|-------|--------|-----------------|
| 1 | Log only | Songbird, squirrel |
| 2 | Spotlight | Unknown bird near coop |
| 3 | Spotlight + audio alarm | Hawk confirmed |
| 4 | Spotlight + siren + audio | Bobcat, coyote |

- **Per-species rules** — hawks get spotlight + audio, bobcats get the full siren
- **Cooldowns** — 5 min between activations per species (neighbor-friendly)
- **Effectiveness tracking** — monitors if the animal leaves within 60 s of deterrent

## PTZ Patrol & Sweep

The Reolink PTZ camera (house-yard) runs a continuous sweep patrol across the full property. The nesting-box camera is fixed and has no PTZ capability.

- **Sweep patrol** (v2.2.0): continuous serpentine raster scan — pans the full range, shifts tilt, reverses direction. Replaces the original preset-hopping approach for more thorough coverage.
- **Dead zone support**: configurable pan range to skip (e.g., the camera's own mounting point)
- **Pause-on-predator**: patrol halts during active deterrence, resumes after

## eBird Early Warning

Polls the Cornell Lab eBird API every 30 min during hawk hours (8 am – 4 pm). Alerts when HIGH/MEDIUM threat raptors are reported within 15 km. Gives 15–60 minutes advance warning before hawks show up on camera.

## Intelligence Reports

- Daily summaries: species breakdown, predator visits, deterrent success rates
- Exported as JSON + Markdown to `data/exports/`
- Hourly activity heatmaps and 7-day trend comparisons
- Queryable via REST API

## REST API

```
GET  /api/v1/status                        Service health
GET  /api/v1/summary/today                 Today's report
GET  /api/v1/summary/{date}                Report for any date
GET  /api/v1/detections?class=hawk&days=7  Query detections
GET  /api/v1/tracks?predator=true&days=30  Query animal visits
GET  /api/v1/patterns/{class_name}         Species activity patterns
GET  /api/v1/deterrents/effectiveness      Deterrent success rates
GET  /api/v1/ebird/recent                  Regional raptor sightings
POST /api/v1/cameras/{id}/ptz              Control PTZ
POST /api/v1/cameras/{id}/spotlight        Toggle spotlight
POST /api/v1/cameras/{id}/siren            Trigger siren
```

## Tech Stack

- **Python 3.13** (Homebrew)
- **OpenCV** — RTSP stream capture and frame processing
- **ultralytics/YOLOv8** — object detection (MPS acceleration on Apple Silicon)
- **reolink_aio** — Reolink camera control (PTZ, spotlight, siren)
- **FastAPI + Uvicorn** — web dashboard + REST API
- **SQLite** — structured detection/track/alert storage (WAL mode)
- **Discord webhook** — alert delivery
- **eBird API** — regional raptor early warning
